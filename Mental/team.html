Tech Stack

Our team used a mix of powerful and accessible tools to build Mindful Garden.

  • Pygame — The core game engine powering our interactive garden and gameplay mechanics.
  • Pixel Art — Handcrafted pixel art for charming bunnies, trees, and garden elements.
  • GitHub — Version control and collaborative coding to keep the project organized and everyone on the same page.
  • Visual Studio Code — Our primary code editor, making coding smoother and more efficient.
